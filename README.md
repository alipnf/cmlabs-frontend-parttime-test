# cmlabs-frontend-parttime-test

Repository ini berisi hasil pengerjaan technical test frontend CMLabs.

## Live Demo

- Vercel : [https://cmlabs-frontend-parttime-test-three.vercel.app/ingredients](https://cmlabs-frontend-parttime-test-three.vercel.app/ingredients)

## Requirements

- Node.js 18 atau lebih baru (disarankan Node.js 20 LTS)
- npm (bawaan Node.js) atau bun

## Cara Menjalankan Project

### 1) Clone repository

```bash
git clone https://github.com/alipnf/cmlabs-frontend-parttime-test.git
cd cmlabs-frontend-parttime-test
```

### 2) Install dependency

Menggunakan npm:

```bash
npm install
```

Atau menggunakan bun:

```bash
bun install
```

### 3) Jalankan mode development

Menggunakan npm:

```bash
npm run dev
```

Atau menggunakan bun:

```bash
bun run dev
```

Buka browser ke `http://localhost:3000`.

## Menjalankan Build Production

Menggunakan npm:

```bash
npm run build
npm run start
```

Atau menggunakan bun:

```bash
bun run build
bun run start
```
